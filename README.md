# Coin price tracker

Este proyecto fue creado utilizando Next.js consumiendo la api publica de [CoinGecko](https://www.coingecko.com/es "CoinGecko") para mostrar el valor (en las ultimas 24hs) de las top 10 cryptos, tambien posee una pesta√±a de Contact Us, donde utilizando Nodemailer un usuario puede contactarnos enviandonos un mensaje a nuestro gmail sin salir de la pagina.

## Comenzando

Para usar esta aplicacion lo unico que debes hacer es clonar este repositorio:

    git clone https://github.com/juanechenique96/coin-price-tracker.git
    yarn
    yarn dev

(Aclaracion: en el directorio /pages/api/contact.js reemplazar process.env.TO por 'tumail@gmail.com')
